# C++ Practice Programs

This repository contains simple C++ programs created for practice while learning the basics of programming.

## Programs Included
- Addition of two numbers entered by the user
- Checking if a number is positive or negative
- Determining a student's grade based on the mark
- Checking if a number is between 1 and 100
- Example of using switch statement

## Concepts Used
- Variables
- User input and output (cin / cout)
- If-else conditions
- Logical operators
- Switch statement
